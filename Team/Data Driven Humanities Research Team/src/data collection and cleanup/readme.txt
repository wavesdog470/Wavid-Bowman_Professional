Purpose: clean XML's with different formating into a data format for easy parsing.
How to use:
	1. Scrape xmls wanted using scraper.py for this project, other scraper if from different database
	2. run clean_xml.py on those using the template.xml as the template file
	3. run findDiv.py and runDiv.py. this will output the cleaned XMLs (check bin)
	4. Use updatedmakecsv.py on the output cleaned xmls in order to create your csv with all the data for parsing
